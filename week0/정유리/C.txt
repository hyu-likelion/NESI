- 1byte = 8bits(bit :binary digit)
- 그냥 %d로 출력하면 hexdecimal, octal 상관없이 decimal로 출력됨
- %x : hexdecimal, %o: octal 로 출력 / X를 대문자로 바꾸면 출력값도 대문자로 나옴
- int에서 가장 왼쪽 bit는 부호를 나타냄(1이면 -)
- 음의 정수 representaion: 절대값이 같은 양의 정수의 bit를 전부 flip한 후 1을 더함
 ex) +5 : 00000101 -> flip : 11111010 -> +1 : 11111011 = -5
